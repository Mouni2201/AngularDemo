ngModel with textbox, checkbox and dropdown


ngModel:

The ngModel directive in Angular facilitates two-way data binding, connecting form input elements like textboxes, checkboxes, and dropdowns to properties in your component's TypeScript class.

TextBox

Value in Component: {{ username }}


In this example, the input field will display the value of the username property. If the user types in the textbox, the username property in the component will not change.

checkbox

Subscribed?

Status: {{ isSubscribed ? 'Yes' : 'No' }}

For a checkbox, [ngModel] is used to set the initial checked state based on a component property. User clicks will not update the component's property.


dropdown

Selected City: {{ selectedCity }}

The dropdown will be initialized with "Banglore" selected. Choosing a different city from the list will not update the selectedCity property.


✅ Key Point: [ngModel] = one-way binding (component → view only). No change from view → component.
